                   FONTLOG for the Deiphobus fonts

This file provides detailed information on the Deiphobus Font Software. This information should be distributed along with the Deiphobus fonts and any derivative works.


Basic Font Information

Deiphobus is an all caps geometric display typeface with bold blocky shapes. The initial design is for Latin script but can easily be extended to cover Greek and Cyrillic scripts.

This release supports the Unicode block Basic Latin / the US-ASCII character set.
